# 🧮 Calculadora en Python

Este proyecto contiene 3 versiones de una calculadora hecha en Python:

* `suma.py` → solo suma
* `version_dos.py` → suma y resta
* `version_tres.py` → suma, resta y división

---

## 🚀 Cómo usarlo (PASO A PASO)

### 1. Abrir la terminal

---

### 2. Entrar a la carpeta del proyecto

```bash
cd programa-suma
```

---

### 3. Verificar archivos (opcional)

```bash
ls
```

Debe aparecer:

```
suma.py
version_dos.py
version_tres.py
```

---

## ▶️ Ejecutar programas

### 🥇 Suma

```bash
python3 suma.py
```

---

### 🥈 Suma y resta

```bash
python3 version_dos.py
```

---

### 🥉 Suma, resta y división

```bash
python3 version_tres.py
```

---

## 🧪 Ejemplo de uso

```
¿Qué quieres hacer?
1. Sumar
2. Restar
3. Dividir

Elige (1, 2 o 3): 3
Primer número: 10
Segundo número: 2

Resultado: 5.0
```

---

## ⚠️ Nota importante

* No se puede dividir entre 0
* Asegúrate de estar dentro de la carpeta correcta (`programa-suma`)

---

## 🧠 Requisitos

* Tener Python 3 instalado

Verificar con:

```bash
python3 --version
```
